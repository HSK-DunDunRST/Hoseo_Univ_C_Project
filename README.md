# Hoseo_Univ_C_Projects

호서대학교 C 프로그래밍 실습 코드 모음입니다. 학기·주차별로 정리되어 있으며, 각 주차 폴더에 해당 날짜의 소스 파일들이 들어 있습니다.

## 디렉토리
- Project_25_1G: 1학기 실습 코드 (Week_01 ~ Week_13)
- Project_25_2G: 2학기 실습 코드 (Week_01 ~ Week_14)

## 파일 이름 규칙
- `Source_YY_MM_DD_X.c` 형태로 날짜와 순서를 나타냅니다.
- 동일한 날짜에 여러 예제가 있을 경우 끝의 번호(X)로 구분됩니다.

## 빌드 예시
```sh
gcc -Wall -Wextra -std=c11 -o main Source_25_12_04.c
./main
```

## 참고
- 주차별 폴더 안에서 각 소스를 바로 빌드해 실행하면 됩니다.
